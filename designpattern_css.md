# Design Pattern – Quiz App

## Inspiration
Dark trivia board game feel – focused and clean with colorful accents.

## Colors
- Page background: #0d1b2a
- Cards and containers: #1a2a3a
- Accent / buttons: #e94560
- Button hover: #c73652
- Main text: #f0f0f0
- Muted text and labels: #aac4e0
- Borders: #2e4a6a

## Typography
- Font: Segoe UI, sans-serif
- Headings: 2rem
- Body text: 1rem
- Labels: 0.9rem

## Spacing
- Card padding: 2rem (desktop), 1.2rem (mobile)
- Gap between form fields: 1.5rem
- Max content width: 700px centered

## Border radius
- Cards: 12px
- Buttons and inputs: 6–8px

## Buttons
- Primary: #e94560 background, white text
- Hover: #c73652
- Delete: transparent with #e94560 border

## Responsive
- Mobile breakpoint: 600px
- Stacked layouts on smaller screens
